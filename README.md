# UW CS Collaboration Graph

Interactive network of **UW CS researchers and their collaborators**, built from Semantic Scholar data.

- Nodes = researchers & coauthors  
- Edge thickness = number of coauthored papers  
- Search, click, and inspect collaborators  
- Hosted live on GitHub pages

## Notes

- The live page is running the graph generated on 1/31/2026 with the professors sourced from https://www.cs.washington.edu/people/faculty-members/
- To get a more recent version (or on a different set of academic) you may:
    - Add your desired researchers to "researchers.txt"
    - Run "build_graph.py"
    - Load "index.html" to view the graph
